1. Introduction
This test utilized ABACUS to perform the SCF calculations on a series of alloy systems (structures are downloaded from material project) containing various elements. The PW (plane wave) base set was used, with ecutwfc set to 100 Ry and kspacing set to 0.2 1/Bohr. Smearing was set to mp with sigma 0.01 Ry, nbands used the default value, and scf_thr was set to 1e-6, and the symmetry was turned off. The results were compared with those of QE (link), which utilized the same pseudopotential and parameter settings. The main comparisons included the energy per atom, the maximum force deviation, the maximum virtual deviation, and the maximum force deviation.
