1. Introduction
This test utilized ABACUS to perform the SCF calculations on a series of alloy systems (structures are downloaded from material project) containing various elements. The plane wave basis set was used, with 'ecutwfc' set to 100 Ry and 'kspacing' set to 0.2 1/Bohr. The 'smearing_method' was set to 'mp' with 'smearing_sigma' 0.01 Ry, 'nbands' used the default value (= max(1.2 * occupied_bands, occupied_bands + 10)), and 'scf_thr' was set to 1e-6, and the symmetry was turned off ('symemtry' = 0). Additionally, the values of 'pseudo_rcut' and 'pseudo_mesh' was set to 10 and 1, respectively, which ensures that the cut-off for radial integration of pseudopotentials and the mesh are consistent with those in Quantum Espresso (QE). The results were compared with those of QE v.7.0 (job link), which utilized the same pseudopotential and parameter settings. The main comparisons included the energy per atom, the maximum force deviation, the maximum virtual deviation, and the maximum force deviation.
