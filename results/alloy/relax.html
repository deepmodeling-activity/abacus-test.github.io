1. Introduction
This test primarily evaluates the accuracy of ABACUS in structural optimization by performing cell-relax calculations on a series of alloy systems, whose structures are downloaded from the Material Project), as well as on the face-centered cubic (fcc), body-centered cubic (bcc), and hexagonal close-packed (hcp) cells of single elements. The PW (plane wave) base set was used, with ecutwfc set to 100 Ry and kspacing set to 0.1 1/Bohr. The convergence criteria for configuration optimization are set to force_thr less than 0.001 Ry/Bohr and stress_thr less than 0.5 kbar. Smearing was set to mp with sigma 0.01 Ry, nbands used the default value (= max(1.2 * occupied_bands, occupied_bands + 10)), and scf_thr was set to 1e-7, and the symmetry was turned off. The results were compared with those of QE (7.0), which utilized the same pseudopotential and parameter settings. At the same time, VASP (5.4.4) was also used to optimize the configuration of these systems, with the ENCUT set to 650 eV and the KSPACING set to 0.189 1/A.
