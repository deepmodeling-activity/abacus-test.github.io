1. Introduction
This test calculated the equation of state (EOS) by using ABACUS, QE, and VASP on a series of alloy systems (downloaded from the Material Project), as well as on the face-centered cubic (fcc), body-centered cubic (bcc), and hexagonal close-packed (hcp) cells of single elements. To estimate the difference of EOS between ABACUS and QE/VASP, the Δ values (reference: 10.1038/s42254-023-00655-3 and 10.1080/10408436.2013.772503) were calculated: Δ = < √(∫(EABACUS(V) - EQE/VASP(V))^2)dV / ΔV) >.
2. Results
For each system, a uniformly spaced 7-point EOS (up to ± 6%, with a step of 2%) was calculated. It is a cell relax calculation but with fixed volume. For ABACUS, set calculation to cell-relax, and set fixed_axes to volume in the INPUT file. For QE, set calculation to vc-relax, and set cell_dofree to shape in the input file. For VASP, set ISIF to 4 in the INCAR file. The energy cutoff is 100 Ry for ABACUS/QE and 650 eV for VASP, and the kspacing is 0.1 Bohr-1 for ABACUS/QE and 0.189 Å-1 for VASP. The convergence criteria of SCF are set to 1e-6 (scf_thr in ABACUS, and conv_thr in QE, and EDIFF in VASP). Symmetry analysis is closed for all calculations (symmetry=0 for ABACUS, nosym=.true. for QE, and ISYM=0 for VASP). ABACUS and QE use the same pseudopotential (please check the detail files in datasets), and VASP use the recommended PAW pseudopotential. The results are shown in Table 1. Based on the volume and energy of different EOS structures, we can fit the third-order Birch-Murnaghan equation by using least-squared method.
