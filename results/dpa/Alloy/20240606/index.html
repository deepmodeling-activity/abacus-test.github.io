1. Introduction
We conducted a series of tests on ABACUS for alloy calculations, aimed at evaluating the property prediction capability and accuracy of ABACUS's plane wave method on alloy systems. This includes: 1. Prediction of elastic modulus properties: The elastic moduli of 53 common alloy elements were calculated using ABACUS's plane wave (PW) basis set combined with dojo and pd04 pseudopotentials, and compared with experimental values; 2. Accuracy verification of plane wave calculation results on DCU platform: Several multicomponent alloy systems were selected for single-point calculations with the PW basis set, and the results were compared with those obtained using Quantum Espresso (QE) under the same parameter settings.
2. Elastic Modulus
We used ABACUS combined with pd04 and sg15 pseudopotentials (additional tests with Rappe pseudopotentials for some elements) to calculate the elastic modulus of 53 common elements in alloys.
All calculations used PW basis, and the value of ecutwfc is set to the converged value tested by the APNS (for tasks where ecutwfc exceeds 100Ry, additional calculations with ecutwfc set to 100Ry were performed), kspacing is set to 0.08 1/Bohr (for Al, kspacing=0.06/0.032 1/Bohr are also tested). The calculation of the elastic modulus was performed using APEX.

We also provide experimental values (Expt, in black), and color the results that closest to the Expt value in blue. Some results are not listed due to the difficulty in convergence.
The test is named as "{element_name}#{pseudopotential}#{ecutwfc}#{kspacing}".
3. Accuracy of Plane Wave Calculation
We selected several multicomponent alloy systems for single-point calculations with the PW basis set on Sugon DCU platform, and compared the results with those obtained using Quantum Espresso (QE) under the same parameter settings (performed on CPU platform). The absolute energy difference per atom (DE, meV/atom), maximum absolute force difference (MADF, eV/Angstrom), and maximum absolute virial difference (MADV, meV/atom) are calculated and compared.
